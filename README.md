# Machine Learning Algorithms

This repository is a growing collection of machine learning algorithms implemented from scratch as I learn the foundations of machine learning.

The goal is to understand the core ideas behind each algorithm, including how models make predictions, how loss functions measure error, and how optimization methods such as gradient descent update model parameters.

## Planned Implementations

1. Linear Regression
2. Logistic Regression
3. Classification Models
4. Neural Networks

## Current Progress

### Linear Regression

Implemented linear regression from scratch using NumPy and gradient descent.

The model is trained and evaluated on the scikit-learn diabetes dataset. I compare training loss over time and evaluate final performance using mean squared error and root mean squared error.

## Topics Covered

- Data loading and preprocessing
- Train/test splits
- Feature standardization
- Loss functions
- Gradient descent
- Model evaluation
- Visualization of predictions
